# 💼 ZapPay Dashboard

ZapPay Dashboard is the admin interface for managing payments, users, and invoices. This interactive UI allows admins to configure settings, monitor transactions, and manage user accounts.

---

## 🎨 Key Features

- 📊 **Transaction Monitoring**: View real-time transaction statuses.
- 🧾 **Invoice & Payment Link Management**: Generate and manage invoices and links.
- 👥 **User Management**: Access social logins and user data.
- 💻 **Cross-Chain Support**: Dashboard that works across multiple blockchains.

---

## 🖥️ Tech Stack

- **React**: Frontend framework for a responsive UI
- **TailwindCSS**: For a clean and modern design
- **Particle Network**: Social login and wallet integration
- **REST API**: Communicates with ZapPay backend

---
